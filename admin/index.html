Admin Portal

Notice

The full admin functionality requires server configuration. Please contact your hosting provider to enable PHP sessions and MySQL database access.

This is a static admin portal for KOSMO Foundation website management. Due to hosting limitations, interactive content management features are currently unavailable.

Setup Instructions

  1. Enable PHP session support on your hosting account
  2. Configure MySQL database access
  3. Run the database setup script (db_setup.php)
  4. Refresh this page to access the full admin interface

Website Structure

The KOSMO Foundation website includes:

  • 14 Program pages with bilingual content
  • Gallery with categorized images
  • Donation system with KakaoPay integration
  • Contact form for inquiries
  • About page with foundation information

File Structure

Main files and directories:

  • index.php - Homepage
  • programs.php - Programs listing
  • program.php - Individual program details
  • donate.php - Donation page
  • gallery.php - Photo gallery
  • about.php - About the foundation
  • assets/ - Images and static files
